# Weakest Precondition Calculator (WPF)

## Описание проекта

Weakest Precondition Calculator — это визуальное приложение на C# WPF для вычисления слабейшего предусловия (weakest precondition) в формальной верификации программ. Приложение позволяет "протаскивать" постусловие через программу в обратном порядке — от конца к началу — демонстрируя пошаговые преобразования и финальную триаду Хоара.

### Основные возможности:
- Расчет слабейшего предусловия для последовательности операторов
- Обработка условных операторов if-else
- Поддержка операций присваивания с арифметическими выражениями
- Автоматическое добавление условий определенности (деление на ноль, корень из отрицательного)
- Пошаговая трассировка вычислений
- Генерация триады Хоара
- Готовые примеры для быстрого старта
- Русскоязычный интерфейс с подробной справкой

## Архитектура проекта

### Back-end компоненты:
- **WeakestPreconditionCalculator** - основной вычислительный движок
- **WPResult** - контейнер для результатов расчета
- Правила преобразования:
  - Присваивание: `wp(x := e, R) = R[x → e]`
  - Последовательность: `wp(S1; S2, R) = wp(S1, wp(S2, R))`
  - Ветвление: `wp(if B then S1 else S2, R) = (B ∧ wp(S1,R)) ∨ (¬B ∧ wp(S2,R))`

### Front-end компоненты:
- **MainWindow** - главное окно приложения
- **HelpWindow** - окно справки
- WPF-интерфейс с группами элементов:
  - Ввод постусловия и описания
  - Редактор кода программы
  - Отображение пошаговой трассировки
  - Вывод результатов
  - Кнопки управления и справки

### Тестирование:
- Набор xUnit тестов для проверки корректности вычислений
- Тесты покрывают основные сценарии использования
- Проверка условий определенности и обработки ошибок

## Установка и запуск

### Сборка из исходного кода:
```bash
# Клонирование репозитория
git clone https://github.com/your-username/wp-calculator.git
cd wp-calculator

# Сборка проекта
dotnet build

# Запуск приложения
dotnet run --project WPCalculator
```

### Запуск тестов:
```bash
dotnet test WPCalculator.Tests
```

### Готовая сборка:
Скачайте последний релиз и запустите `WPCalculator.exe`

## Использование

1. Введите постусловие в формальном виде (например: `max > 100`)
2. Добавьте человеко-читаемое описание
3. Введите код программы в редакторе
4. Нажмите "Рассчитать WP" для вычисления
5. Просмотрите пошаговую трассировку и результат
6. Используйте "Показать триаду Хоара" для финального представления

## Примеры использования

Приложение включает три готовых примера:

- **Квадратное уравнение** - работа с математическими операциями
- **Последовательность присваиваний** - линейные преобразования
